# Ideation Archive

This directory holds research notes, opportunity scans, proof slices, and build plans that informed Kontour AI's product direction but are not the current public thesis.

Current product positioning lives in `../product-line-vision.md`.

## Lifecycle

- Store exploratory work here when it is useful context but not active product documentation.
- Promote durable decisions into `../product-line-vision.md` or product repos.
- Keep opportunity entries conceptual. Do not name prototype repositories or customer-specific vertical products.
- Revisit this archive periodically and remove notes once their decisions have been implemented elsewhere or no longer guide current planning.
